# ShiftGeno project page

This directory contains the static GitHub Pages site for ShiftGeno.

- `index.html`: page content and metadata
- `assets/css/style.css`: responsive visual design
- `assets/js/main.js`: navigation, reveal effects, figure lightbox, and citation copying
- `assets/images/`: figures extracted from the project manuscript

The site is designed to be served from `https://mapengsen.github.io/ShiftGeno/`.
